<h2 align="center"> Technologies</h2>

Project is created with:

- React: 16.13.0
- Bootstrap 4: (4.4.1)
- Auth0 Authentication (auth0-js): 9.13.1
- SASS: 4.13.1
- Contentful: 7.14.3
- React Icons: 3.9.0
- React Lottie: 1.2.3
- Formspree (backend email service)
